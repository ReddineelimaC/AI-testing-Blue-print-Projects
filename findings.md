# Findings

- Objective: Create a local LLM testcase generator using Ollama API.
- Constraints: Must run locally, use Ollama, and follow the provided prompt.

## Discovery Questions (Phase 1.1)
- **North Star:** Local LLM testcase generator based on User input with a proper code-stored Template, using Ollama API (open source model llama3.2).
- **Integrations:** Ollama (local API).
- **Source of Truth:** N/A (Stateless or in-memory chat session).
- **Delivery Payload:** A UI chat interface where the user enters input, and Ollama generates testcases.
- **Behavioral Rules:** User enters input -> output is generated by local LLM in Ollama using the template.

## Research (Phase 1.3)
- GitHub repo examples: `pankajsarmah10/ai-testdata-generator` (Selenium + Ollama), `matinict/MyLLM` (Langchain + Ollama + Gemini).
- Framework options: Using raw Python `requests` to the local Ollama API (e.g. `http://localhost:11434/api/generate`) or building around a framework like Langchain/LlamaIndex.
- UI implementation: We will build a web-based Chat UI using HTML/JS/CSS that communicates with the local Ollama instance at `http://localhost:11434/api/chat`.
